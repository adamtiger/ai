About the Site

Machine learning is one of the fastest growing area of science. This site gathers and puts into context a lot of information about Ai included: ethical questions, philosophycal questions. The main focus is on machine learning and especially on algorithms.

The tab ML contains articles. Each article has a tag with which you can filter them. The main topics covered so far are: reinforcement learning, genereal view on ai, deep learning etc.
